Git add

git add put file or folder name here

git add . put file or folder name here

This will add all folders and files to the staged area

git add *. put file name here.html

This will add all html file to the staged area

git add -A put file or folder name here

To add and track all files to the staged area

Git diff Show difference in modified files

git diff

This will add all folders and files to the staged area

git diff put file or folder name here

This will show differences between modified and repo

git diff --staged

This will show difference between staged and repo

git mv put file or folder name here, put the new of folder here

renames file

git mv put file or folder name here, put the new folder name here/, put the name or rename of folder or file here

moves a file

Git remove

git rm --cached put file name here

This will remove a file from a staged area

git rm --cached -r put a folder path name here

This will remove a folder from a staged area

Git commit

git commit put file or folder name here

This will commit any staged files and folders to the repository

You will have to remove a hashtag and right your message then hit esc then put :wq

git commit -m "Put statement to save"

To commit changes/updates/additions to repository

git commit -a -m "Put statement to save"

To add and commit changes/updates/additions to repository

Git ignore

create a file named .gitignore

Then put the names of the files or folders you do not want to be tracked inside the .gitignore file

This will keep files and folders from being tracked

*.html

Will keep all html file untracked

/put folder name here

This will keep this folder untracked

put file name here

This will keep this file untracked

Git branch

git branch

This will let you view a list of branches:

git branch put a branch name

This will create a branch

git checkout -f name of branch

This will reject latest changes

git checkout -b name of branch

To create a feature branch:

git branch -d name of branch

To delete a feature branch that is no longer needed and has been merged to master already:

git branch -D name of branch

To delete a feature branch that is no longer needed but has NOT been merged to master:

git checkout name of branch

To move to already created feature branch:

git merge name of branch

To merge the changes in the feature branch to the master branch first switch to master branch:

git remote -v
